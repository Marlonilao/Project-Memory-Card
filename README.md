# Pokémon Memory Card Game

This project is part of the React course from [The Odin Project](https://www.theodinproject.com?utm_source=chatgpt.com).

## Description

A Pokémon-themed memory card game built with React where players must avoid clicking the same Pokémon twice. Each successful unique click increases the score, while clicking a previously selected Pokémon resets the current score.

The Pokémon cards are shuffled after every turn to increase the difficulty and challenge the player's memory.

This project focuses on practicing core React concepts such as state management, props, component composition, effects, and fetching data from external APIs.

## Technologies Used

- React
- JavaScript (ES6+)
- CSS
- Mantine (UI Library)
- PokéAPI
- Vitest
- React Testing Library

## Features

- Dynamic card shuffling
- Score tracking
- Best score tracking
- Data fetching from an external API

## Testing

Component tests written with Vitest and React Testing Library covering
loading, scoring, win/loss states, and error handling.
